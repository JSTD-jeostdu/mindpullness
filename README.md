# Mindpullness 🪁

**마음챙김, 마음당김 워크북** — 내 마음을 챙기고 당기는 개인용 셀프케어 웹앱.

> mindfulness가 아니라 **mind + pull + ness**.
> 마음을 건강한 상태로 붙잡아, 당겨 놓는다는 뜻입니다.

## 무엇을 하는 앱인가

- ✏️ **생각기록지** — 상황·생각·감정·신체·행동을 분리해 기록하고, 탈중심화 질문으로 마음당김 (감정 강도 전→후 자동 비교)
- 🤖 **클로드 코치 연동** — 버튼 하나로 기록+코치 지침이 프롬프트로 복사되어, Claude가 1:1 코치 역할
- 📖 **4주 28강 배움 코스** — CBT·ACT 기반 하루 5분 마이크로 레슨
- 🌬️ **호흡·마음챙김 훈련** — 박스 호흡, 4·7·8 호흡, 생각 흘려보내기, 바디 스캔
- 📊 **마음 체크** — PHQ-9·GAD-7 자가 점검과 추이 그래프
- ☁️ **선택적 클라우드 동기화** — Firebase config를 넣으면 Google 로그인으로 기기 간 동기화
